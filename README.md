# Curveapp Gym Vendor Panel

The web dashboard for gym vendors on the Curve platform. Allows gym owners to manage memberships, subscription plans, offers, and member activity.

## Tech Stack

- **Framework:** Angular 15
- **Language:** TypeScript
- **Styling:** CSS / SCSS
- **Build Tool:** Angular CLI

## Getting Started

```bash
npm install
ng serve
```

Navigate to `http://localhost:4200/`.

## Build

```bash
ng build
```

Build artifacts are stored in the `dist/` directory.
